ProcessNotificationSequenceResponseParameters

https://demis.rki.de/fhir/StructureDefinition/ProcessNotificationSequenceResponseParameters

Der ProcessNotificationSequenceResponseParameter ist der Ausgabeparameter für die ProcessNotificationSequence-Operation. Er beinhaltet die von DEMIS erstellte und ausgegebene IGS-Id (TransactionID), die Meldungs-Id (NotificationID) der ursprünglich vom Einsender der Probe erstellten Meldung (SubmitterGeneratedNotificationID), die laboreigene Id der Sequenz (LabSequenceID) und das Ergebnis der Verarbeitung (OperationOutcome).

If parameter is a whole resource

Definition

If the parameter is a whole resource.

Comments

When resolving references in resources, the operation definition may specify how references may be resolved between parameters. If a reference cannot be resolved between the parameters, the application should fall back to it's general resource resolution methods.
