Primer-Scheme (PrimerSubstance)

https://demis.rki.de/fhir/StructureDefinition/PrimerSubstance

Die PrimerSubstance dient der Angabe des Primer-Scheme (z.B. ARTICv4.1), das zur Amplicon-Sequenzierung verwendet wurde. Zusätzlich kann ein Link auf die hinterlegten Sequenzen angegeben werden (z.B. github.com/...). Die Ressource wird pro Primer-Scheme einmal verwendet.
